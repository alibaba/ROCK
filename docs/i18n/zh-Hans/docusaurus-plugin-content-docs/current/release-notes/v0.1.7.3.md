# V0.1.7.3

## 更新日期
2025.11.19

## SDK
### New Features
* `Sandbox.arun()` 方法新增: `response_limited_bytes_in_nohup` 参数，解决response过长导致的请求超时问题。具体参考[Sandbox SDK Reference](../references/sandbox.md)
* `Sandbox`新增`read_file_by_line_range`: 需要注意该方法无法保证与原始文件内容完全一致，可能会丢失换行符或其他信息。

### Upgrades
* Wheel包依赖项优化，降低与其他项目的冲突概率
